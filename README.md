# 孔乙己

#### 项目介绍
主要对象是爬booktxt网上的小说。
将来可以添加配置文件，把爬的规则独立出来。通过配置可以爬不同的网站小说了。
目前只实现了booktxt的爬取。
线程数20个，以后可以更改为可配置。

#### 更新记录
完善拆分为2个辅助类。现在可以通过配置kung_ke.json文件来维护不同网站的规则了。
使用的时候需要显示的输入是哪个网站的，输入域名即可。
当没有kung_ke.json文件或者文件为空，将从ini文件中的地址进行下载json配置文件。
